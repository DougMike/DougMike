## 🚀 Projetos em Destaque

### 💰 [Financiapp](https://github.com/DougMike/financiapp)
> Aplicação full stack de gestão financeira pessoal

- **Back-end:** ASP.NET Core 8 com Clean Architecture (Domain / Application / Infrastructure / Server)
- **Front-end:** Angular 18 standalone, Bootstrap 5, SCSS com design system próprio
- **Features:** Controle de receitas e despesas, categorias, paginação no banco (EF Core `Skip/Take`), JWT Authentication, suporte a múltiplos idiomas (pt-BR / en)
- **Boas práticas:** SOLID, DRY, Repository Pattern, DTOs, Interceptors

---

### 🏨 [HotelReservation](https://github.com/DougMike/HotelReservation)
> API REST para sistema de reservas de hotel

- **Stack:** ASP.NET Core 6 · EF Core · SQL Server · FluentValidation · Swagger
- **Features:** CRUD de reservas com regras de negócio (máx. 3 dias de estadia, reserva com até 30 dias de antecedência), gestão de hóspedes e quartos
- **Arquitetura:** Domain → Service → Repository, Injeção de Dependência, padrão de validação genérico com `AbstractValidator<T>`

---

### 🎸 [Luthieria](https://github.com/DougMike/Luthieria)
> API para gerenciamento de instrumentos musicais em uma luthieria

- **Stack:** ASP.NET Core 3.1 · EF Core · SQL Server
- **Destaque:** BaseRepository genérico com `IBaseRepository<T>`, suporte a guitarra, baixo e violão

---

### 👗 [Refah](https://github.com/DougMike/refah)

- **Stack:** HTML5 · CSS3 · JavaScript · Bootstrap 5
- **Features:** Layout responsivo, galeria com lightbox, slider de depoimentos (Swiper), animações AOS, formulário de contato
